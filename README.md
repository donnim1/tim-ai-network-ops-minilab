# TIM AI Network Ops MiniLab

Telecom AI prototype simulating autonomous network operations.

## Features

- Telecom network telemetry simulator
- Anomaly detection with ML
- Graph DB topology modeling
- RAG assistant for network troubleshooting
- FastAPI backend
- Docker-ready architecture

## Tech Stack

Python, FastAPI, Neo4j, ChromaDB, LangChain, Scikit-learn, Docker, GitHub Actions

## Use Case

AI-driven network monitoring for telecom operators such as TIM.
